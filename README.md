# 👀 Show / Hide Element

مشروع بسيط باستخدام JavaScript للتحكم في إظهار وإخفاء عنصر داخل الصفحة.

---

## 🚀 فكرة المشروع

التطبيق يحتوي على:

- نص أو عنصر
- زر "إخفاء" لإخفاء العنصر
- زر "إظهار" لإظهار العنصر مرة أخرى

## 🛠️ التقنيات المستخدمة

- HTML5
- CSS3

## 🎯 ماذا تعلمت من المشروع؟

- التحكم في CSS باستخدام JavaScript
- استخدام `style.display`
- الفرق بين `none` و `block`
- التعامل مع الأحداث `onclick`
